Dependency:

PCL
Glog
GTSAM
Ceres
#OpenCV


Install:

cd ~/catkin_ws/src
git clone git@github.com:TongxingJin/lab_slam.git
cd ..
catkin_make


Usage:

roslaunch lab_slam run.launch
